## About the Project

**CrackIT AI Interview Coach** is an AI-powered mock interview platform designed to simulate real interview experiences and help users build confidence before facing actual interviews. The platform offers 500+ carefully curated interview questions across five specialized question banks—Data Structures & Algorithms (150), Java (100), DBMS (100), Operating Systems (75), and HR (75)—allowing users to practice a wide range of technical and behavioral topics. With support for both voice and text-based responses, the application leverages Natural Language Processing (NLP) to evaluate answers, generate instant scores, and provide personalized feedback. By tracking performance over time, CrackIT AI enables users to identify areas for improvement, strengthen their communication skills, and approach interviews with greater confidence.

## What the Project Does

*  Secure user registration and login using JWT authentication.
*  Supports both voice-based and text-based interview responses.
*  Conducts AI-powered mock interviews with predefined questions.
*  Uses NLP (spaCy) to analyze and evaluate user responses.
*  Generates interview scores and personalized feedback.
*  Displays interview history and performance through a dashboard.
*  Helps users improve communication and answer quality.
*  Provides a clean and responsive user interface for a smooth interview experience.

##  How It Works

1. Users register or log in securely.
2. The system presents interview questions one at a time.
3. Users respond using either voice or text input.
4. The backend processes and analyzes responses using NLP.
5. The application generates scores and constructive feedback.
6. Users can review previous interview results and track their progress from the dashboard.

##  Tech Stack

* **Frontend:** React.js, HTML, CSS, JavaScript
* **Styling:** Tailwind CSS
* **Backend:** Flask (Python)
* **Database:** SQLite
* **Authentication:** JWT (JSON Web Token)
* **Natural Language Processing:** spaCy
* **API Communication:** REST APIs (Flask-CORS)
* **Environment Management:** Python Dotenv
